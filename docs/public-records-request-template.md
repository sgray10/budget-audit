# Public Records Request Template

Use a neutral, narrow request. Ask for machine-readable records where available.

```text
Subject: Public records request — budget and compensation data

Hello,

I am requesting public records related to [department/body/fund] for fiscal year [year].

Where available, please provide records in their original electronic format, such as Excel, CSV, accounting-system export, or other machine-readable files. If records are only available as PDF, scanned documents, or paper copies, please let me know.

Requested records:

1. Adopted budget and amended budget records for fiscal year [year].
2. Line-item budget detail by fund, department, account/object code, and description.
3. Budget amendment detail, including resolution number, date, fund, account/object code, increase/decrease amount, and stated reason if available.
4. Salary or compensation line-item detail sufficient to identify position title, department, fund, and annual amount where public and legally releasable.
5. Any public investment reports required under county investment policy for [date range].

Please include any data dictionaries, account-code descriptions, fund-code descriptions, or export layouts that explain the records.

I am not requesting confidential information, private personal identifiers, or records exempt from disclosure. If any portion of a record is exempt, please provide the non-exempt portions and cite the applicable exemption for any redaction.

Thank you,
[Name]
```

## Notes

- Keep requests narrow enough to be answerable.
- Ask for native electronic format first.
- Ask for code descriptions and data dictionaries.
- Do not imply wrongdoing in the request.
- Track request date, response date, fees, exemptions, and produced files.
